# Continue.dev

## Installation

```bash
# Global
cp /path/to/product-context/skills/continue/product-context.md ~/.continue/rules/

# Or project-level
mkdir -p .continue/rules
cp /path/to/product-context/skills/continue/product-context.md .continue/rules/
```

> Continue uses `.continue/rules/*.md` format, not SKILL.md.
