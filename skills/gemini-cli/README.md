# Gemini CLI

## Installation

```bash
# Global
ln -s /path/to/product-context/skills/gemini-cli ~/.gemini/skills/product

# Or project-level
cp -r /path/to/product-context/skills/gemini-cli .gemini/skills/product
```

## Usage

```
/product init    # Initialize .product/ directory
/product         # Analyze conversation, update product docs
```
