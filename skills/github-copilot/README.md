# GitHub Copilot

> **Note**: Skills support is experimental (as of Dec 2025)

## Installation

```bash
# Global
cp -r /path/to/product-context/skills/github-copilot ~/.copilot/skills/product

# Or project-level
cp -r /path/to/product-context/skills/github-copilot .github/skills/product
```

Enable `chat.useAgentSkills` in VS Code settings.
