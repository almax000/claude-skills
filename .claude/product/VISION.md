# Vision

## What is this?

A personal collection of Claude Code skills - battle-tested workflows and automation patterns for AI-assisted development.

## Why?

Skills are the entry point to Claude Code customization. Through skills, you can generate:
- Custom agents for specialized tasks
- Hooks for automated workflows
- Rules for coding standards

Instead of sharing complex configurations, share skills that generate them.

## Core Philosophy

**Skills as the single entry point** - Every customization starts with a skill. This keeps things:
- Simple: One thing to learn
- Portable: Easy to share and install
- Generative: Skills create other components

## Value Proposition

| For Whom | Value |
|----------|-------|
| Claude Code newbies | Ready-to-use skills that just work |
| Individual developers | Save time with proven workflows |
| Small teams | Consistent practices across projects |
