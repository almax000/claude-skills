# Vibling 路线图

## 当前阶段
MVP - 验证可视化生成的核心价值

## 里程碑

### Phase 1: MVP (2025-02)
**目标**: 用户能上传音乐，选择模板，导出视频

**核心功能**:
- [x] 音频上传（MP3）
- [x] 音频波形显示
- [ ] 音频频率分析
- [ ] 模板系统架构
- [ ] 3个基础模板
- [ ] 视频导出（WebM）

**技术债务可接受**:
- 只支持MP3格式
- 导出只支持720p
- 无用户账号系统

**完成标准**: 10个用户成功导出视频

### Phase 2: Beta (2025-04)
**目标**: 提升可用性，收集反馈

- [ ] 更多音频格式（WAV, FLAC）
- [ ] 更多模板（10个+）
- [ ] 模板自定义（颜色、文字）
- [ ] 导出质量选项（720p/1080p/4K）
- [ ] 用户反馈收集机制

### Phase 3: 商业化 (2025-06)
**目标**: 验证付费意愿

- [ ] 用户账号系统
- [ ] 免费/付费模板分层
- [ ] 水印移除（付费功能）
- [ ] 支付集成
- [ ] 订阅制定价

### 未来考虑
- AI生成模板
- 模板市场（用户上传模板）
- 移动端支持
- 与音乐平台集成（Spotify, SoundCloud）
- 实时演出模式

## 竞品观察
- Renderforest - 功能多但复杂
- Kapwing - 通用视频工具
- Headliner - 播客可视化
