# 开发路线图

## 当前阶段
原型 - 验证核心玩法

## 里程碑

### Phase 1: 原型 (Week 1-2)
**目标**: 验证核心玩法是否有趣

- [ ] 核心机制实现
- [ ] 最小可玩版本
- [ ] 自己玩测试

**完成标准**: 自己玩10次还觉得有趣

### Phase 2: 垂直切片 (Week 3-4)
**目标**: 完整的第一关/第一个循环

- [ ] 完整游戏循环
- [ ] 基础UI
- [ ] 基础音效
- [ ] 请朋友测试

**完成标准**: 朋友愿意玩完整个关卡

### Phase 3: 内容扩展 (Week 5-8)
**目标**: 足够的内容量

- [ ] [N]个关卡/[N]小时内容
- [ ] 难度曲线调整
- [ ] 更多美术资源
- [ ] 更多音效/音乐

### Phase 4: 打磨 (Week 9-10)
**目标**: 提升整体质量

- [ ] Bug修复
- [ ] 性能优化
- [ ] Juice效果（屏幕震动、粒子等）
- [ ] 教程/新手引导

### Phase 5: 发布准备 (Week 11-12)
**目标**: 准备上线

- [ ] 商店页面
- [ ] 宣传素材
- [ ] 成就系统
- [ ] 数据统计

### 发布后
- [ ] 收集反馈
- [ ] Bug修复
- [ ] 考虑更新/DLC

## 未来考虑
- 移植到其他平台
- 多语言支持
