# BUSINESS.md 格式规范

**版本**: v1
**状态**: 稳定
**类型**: 扩展文件

## 概述

BUSINESS.md 定义商业模式和定价策略，回答"钱从哪来"。

聚焦于商业决策，不涉及支付集成等技术实现（那是 AGENTS.md）。

## 必需字段

```markdown
# 商业模式

## 收入模式
<!-- 怎么赚钱 -->

## 定价策略
<!-- 价格怎么定 -->
```

## 推荐字段

```markdown
## 免费版策略
<!-- 免费版包含什么，限制什么 -->

## 竞品定价分析
<!-- 竞品怎么定价 -->

## 单位经济
<!-- 获客成本、LTV 等 -->
```

## 示例

```markdown
# Vibling 商业模式

## 收入模式
Freemium + 订阅制

## 定价策略

### 免费版
- 3 个基础模板
- 720p 导出
- 带水印
- 每月 5 个视频

### Pro 版 - ¥29/月
- 所有模板
- 1080p 导出
- 无水印
- 无限视频
- 优先支持

### 年付 - ¥249/年（相当于 ¥21/月）
- 所有 Pro 功能
- 送 2 个月

## 免费版策略

### 设计原则
1. **足够好用**: 免费版要能完成核心流程
2. **有升级动力**: 水印和分辨率是升级驱动
3. **不是试用版**: 免费版是永久的

### 限制选择理由
| 限制 | 原因 |
|------|------|
| 水印 | 低成本实现，升级动力强 |
| 720p | 社交媒体够用，但专业场景不够 |
| 5个/月 | 偶尔用户够用，重度用户需升级 |

## 竞品定价分析

| 竞品 | 定价 | 我们的优势 |
|------|------|------------|
| Renderforest | $9.99/月起 | 更专注音乐场景 |
| Kapwing | $16/月 | 更便宜 |
| After Effects | $22.99/月 | 简单太多 |

## 单位经济（目标）

| 指标 | 目标值 |
|------|--------|
| 免费转付费率 | 5% |
| 月付转年付率 | 30% |
| ARPU | ¥25/月 |
| LTV | ¥300 |
| CAC 目标 | < ¥50 |
```

## AI 使用指南

当 AI 读取 BUSINESS.md 时，应该：

1. **功能限制** - 实现时按定价策略限制功能
2. **升级引导** - 免费用户遇到限制时引导升级
3. **文案配合** - 升级文案遵循 BRAND.md 调性
4. **不主动加限制** - 只实现已定义的限制，不自行添加
