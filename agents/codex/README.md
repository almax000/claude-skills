# OpenAI Codex CLI Integration

Product Context 的 OpenAI Codex CLI 集成。

## 安装方式

1. 复制 `AGENTS.md.template` 的内容
2. 添加到项目的 `AGENTS.md` 文件中

## 文件说明

| 文件 | 用途 |
|------|------|
| `AGENTS.md.template` | AGENTS.md 集成模板 |

## 使用效果

Codex CLI 将会：
- 在实现功能前检查 `VISION.md`
- 在写用户面向内容时遵循 `BRAND.md`
- 根据 `ROADMAP.md` 确定优先级
- 在重要进展后更新 `MEMO.md`
