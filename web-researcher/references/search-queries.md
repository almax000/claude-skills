# Search Query Templates

## Technical Research

```
[library/framework] [specific feature] [version] [year]
Example: "Next.js app router middleware 14 2026"

[language] best practices [topic] [year]
Example: "TypeScript best practices error handling 2026"

[framework] vs [framework] comparison [year]
Example: "Prisma vs Drizzle comparison 2026"
```

## Documentation Lookup

```
[library] official documentation [topic]
Example: "React official documentation useEffect cleanup"

[library] API reference [method/function]
Example: "Node.js API reference fs.readFile"
```

## Troubleshooting

```
[exact error message]
Example: "TypeError: Cannot read property 'map' of undefined React"

[library] [error type] [context]
Example: "Next.js 404 error dynamic routes"
```

## Best Practices

```
[topic] best practices [context] [year]
```
