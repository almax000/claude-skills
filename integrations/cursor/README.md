# Cursor Integration

Product Context 的 Cursor 集成。

## 安装方式

1. 复制 `.cursorrules.template` 的内容
2. 添加到项目的 `.cursorrules` 文件中

或者创建 `.cursor/rules/product-context.mdc` 文件。

## 文件说明

| 文件 | 用途 |
|------|------|
| `.cursorrules.template` | Cursor 规则模板 |

## 使用效果

Cursor 将会：
- 在实现功能前参考 `VISION.md`
- 在写文案时遵循 `BRAND.md` 调性
- 根据 `ROADMAP.md` 理解优先级
- 从 `MEMO.md` 恢复上下文
