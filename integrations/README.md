# AI Tool Integrations

Configuration templates for various AI coding assistants.

> **Note**: For Claude Code, use the [skill](/skill/) instead of these templates.

## Available Integrations

| Tool | Directory | Installation |
|------|-----------|--------------|
| Cursor | [cursor/](cursor/) | Add to `.cursorrules` |
| GitHub Copilot | [copilot/](copilot/) | Add to `copilot-instructions.md` |
| OpenAI Codex CLI | [codex/](codex/) | Add to `AGENTS.md` |
| Windsurf | [windsurf/](windsurf/) | Add to `.windsurf/rules` |
| Universal (any AGENTS.md tool) | [universal/](universal/) | Add to `AGENTS.md` |

## File Structure

Each integration directory contains:
- A template file (`.template`) with the configuration to copy
- A `README.md` with installation instructions

## Usage

1. Choose your AI tool from the table above
2. Navigate to the corresponding directory
3. Follow the README instructions to integrate Product Context
